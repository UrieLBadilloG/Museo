# React + Vite

##Actualizar sistema

```
sudo apt update
sudo apt upgrade
```

Instalar Node.js y npm
Si no tienes Node.js y npm instalados, puedes hacerlo ejecutando:

bash

```
sudo apt install -y nodejs npm
```

##Clonar el repositorio

```
git clone https://github.com/tu-usuario/tu-proyecto.git
```

```
cd proyecto
npm install
```

```
npm run dev
```
